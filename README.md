# Flight Booking App

## Tech stack

- **React**
- **Redux Toolkit**
- **TypeScript**
- **SASS**
- **Webpack**
- **ESLint**
- **Prettier**

## Requirements

- Node.js >= 14
- npm >= 6

## Getting started

1. Clone the repository:

    ```bash
    git clone 
    ```

2. Go to the project directory:

    ```bash
    cd flight-booking-app
    ```

3. Install dependencies:

    ```bash
    npm install
    ```

## Scripts

### Run in development mode

```bash
npm start
```

### Starting the project build

```bash
npm run build
```
